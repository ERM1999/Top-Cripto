# 📊 Crypto Dashboard con Flet

Aplicación tipo dashboard desarrollada en Python utilizando Flet para visualizar datos de criptomonedas de forma clara e interactiva.

---

## 🚀 Características

- ✅ Interfaz moderna tipo app móvil
- ✅ Navegación entre múltiples pantallas
- ✅ Datos simulados preparados para API real
- ✅ Sistema de guía para entender la información
- ✅ Cambio de idioma (español / inglés) en la sección informativa
- ✅ Estructura escalable para futuras integraciones (API + Kafka)

---

## 📱 Pantallas

### 🏠 Home
- Precio actual
- Cambio en 24h
- Máximo y mínimo
- Resumen de situación del mercado

### 📊 Estadísticas
- Capitalización
- Volumen 24h
- Posición
- Suministro
- Máximo histórico
- Tendencia y volatilidad
- Sección futura para gráfico

### 📘 Guía
- Explicación de todos los indicadores
- Disponible en español e inglés
- Cambio de idioma dinámico

---
